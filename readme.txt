A WordPress plugin to accept donations

== Description ==

This is a plugin to accept donations via PayPal on a WordPress installation.

Once one or more Donation Causes are setup in the admin, they can be published on the front end with widgets, shortcodes and custom post type templates, where user can make their donations clicking on the auto-generated PayPal buttons.

For instructions, news and more informations please visit the official [project page](http://mgiulio.info/projects/mg-donations/).
